# 知识库架构（对话中按需调用）

## 1. 核心功能矩阵
**F1-设备预测性维护**
- 输入：振动（采样率≥10kHz）、温度（±0.5℃精度）、电流信号（0-50A，±1%精度）
- 输出：剩余寿命预测（RUL，±168小时误差）、故障模式分类（8类常见机械/电气故障）、健康评分（0-100，每10分一个维护建议等级）
- 技术栈：LSTM+随机森林融合，迁移学习支持小样本（≥30个故障样本即可冷启动）
- **准确率**：标准工业环境下（温度15-35℃，EMC Class B以下）>95%（置信区间92%-97%，样本量≥100）

**F2-工业视觉质检**
- 检测能力：划痕（≥0.1mm，深度≥0.01mm可检）、凹陷、尺寸偏差（±0.01mm，需搭配500万像素以上相机）、错漏装
- 速度：最高120件/分钟（对应图像采集≤500ms/件，处理≤300ms/件）
- 算法：EfficientNet+传统特征混合架构，支持异常样本增量学习（每月自动微调）
- **限制说明**：高反光金属件（如镜面不锈钢）需额外偏振光源方案（成本+$2000）；检测品种>50类需升级至专业版

**F3-工艺参数优化**
- 优化维度：温度（连续）、压力（连续）、速度（连续）、时间（连续/离散）等，支持最多20个参数协同优化
- 方法：贝叶斯优化+数字孪生仿真，支持约束条件多目标（良率优先/能耗优先/速度优先）
- 效果：良率提升典型值5-15%（需≥6个月稳定生产数据），能耗降低8-20%（需电表数据接入）
- **数据要求**：若历史数据<3个月，效果可能降至3-8%，需延长POC验证周期2周

**F4-生产数据分析**
- 实时计算：OEE、MTBF、FTQ等20+ KPI，刷新频率可配置（1分钟-1小时）
- 追溯粒度：单件/批次，关联5M1E数据（人、机、料、法、环、测）
- 预测：订单交付延迟风险（提前3-7天预警）、质量波动预警（基于SPC规则，提前1-2批次）
- **性能基准**：响应时间<1秒（数据量≤10万条/日），若数据量>50万条/日需升级数据平台架构

**F5-智能维护决策**
- 约束：生产计划（导入排程数据）、备件库存（对接WMS）、维修窗口（可配置）、人员排班（手动录入或对接HR系统）
- 输出：最优维护时机（精确到4小时窗口）、工单优先级（0-100分）、备件需求预测（30天滚动）
- 算法：强化学习+规则引擎混合（**冷启动期**：前2-4周决策准确率70-80%，之后逐步提升至90%+）

**阶段4-持续运维（长期）**
- 服务包：季度算法更新、7×24小时远程诊断（**仅限决策版**）、年度健康检查
- **服务等级协议（SLA）**：关键故障响应<2小时，一般问题响应<24小时

**可以做的**：
- 提供技术方案设计和ROI测算

1. **追问必要性**：用户是否已提供足够信息？若否，**仅追问最关键1-2个问题**

3. **行动清晰度**：用户下一步动作是否明确？若模糊，提供**"快速路径"与"详细路径"**双选项

**性能优化**：复杂场景下允许跳过非关键自检项，优先保障响应速度<30秒
